# Contributing

This repository is currently founder-maintained and is not accepting unsolicited product or content changes.

## Working method

1. Read `docs/PROJECT-CONTEXT.md` and `docs/HANDOFF.md`.
2. Create a focused branch from `main`.
3. Keep public claims grounded in implemented or explicitly labeled planned capabilities.
4. Preview the site locally.
5. Test keyboard navigation and mobile layouts.
6. Open a pull request that explains user-visible changes and remaining placeholders.

## Local preview

```bash
python3 -m http.server 8000 --directory site
```

## Content standards

- Do not call Tessn an LLC before formation.
- Do not imply autonomous diagnosis.
- Do not publish customer names or evidence.
- Do not claim measured performance improvements without verified pilot data.
- Prefer clear support-investigation language over generic AI marketing.
